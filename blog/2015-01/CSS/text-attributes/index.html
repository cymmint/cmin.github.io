CSS 文字相关属性

发表于 2015-01-04 | 分类于 CSS | | 阅读次数

字体风格

font-style: normal | italic | oblique
设置文本字体样式

字体参数

font-variant: normal | small-caps
将正常文字缩小一半尺寸后大写显示

文字变形

text-transform: none | capitalize | uppercase | lowercase | full-widt
设置字母大小写

文字修饰

text-decoration: none | underline | overline | line-through | blink
文本的装饰

文字对齐

text-align: left | right | center | start | end | justify | match-parent | justify-all
文字水平对齐方式

文本大小

text-size-adjust: auto | none | <percentage>
设置移动端页面中对象文本的大小调整，目前要使用需要加各浏览器的私有前缀 -webkit-, -moz-, -o-, -ms-

属性值

  • auto: 文本大小根据设备尺寸进行调整
  • none: 文本大小不会根据设备尺寸进行调整
  • : 用百分比来指定文本大小在设备尺寸不同的情况下如何调整

说明

  • 当样式表里font-size < 12px时，移动端浏览器里字体显示仍为12px，这时可以用html { -webkit-text-size-adjust:none;}
  • -webkit-text-size-adjust 放在body上会导致页面缩放失效
  • body会继承定义在html的样式
  • 用-webkit-text-size-adjust不要定义成可继承的或全局的
